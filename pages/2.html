2.3 Operators ¶

Operator Description
operator (size_t end) operator (size_t start, size_t end) Same as the slice member function
Comparison operators: <, >, <=, => Perform lexicographic comparison between two strings. Can be used to compare with C-strings as well.
Equality operator: == Tests if two strings are equal (case-sensitive). Can be used to compare with C-strings as well
Concatenation operator: + Concatenates two strings and returns a new string. Can be used to concatenate with C-strings as well.
Concatenation shorthand: += Shorthand for concatenating another string into the current string. Can be used with C-style strings as well.
Assignment operator: = Assign a string to another string. A C-string can also be assigned to a string object.
Type cast operator to std::string_view Cast the string to a std::string_view type
Stream insertion operator: >> Insert the string into the output stream character by character
Stream extraction operator: << Extract the contents of the input stream and insert it into the string object
Item access operator: [] Access individual characters of the string by index value
